# judicial-application-moshe-shamash-land-rights

Generate a formal, court-ready Hebrew judicial application for the Israeli judicial land-registration route, seeking a court order to recognize and register inherited rights from Moshe Shamash to Sarina Shamash in Gush 38049 / Parcel 36 / Subparcel 35.

## Objective

Draft the strongest possible Hebrew judicial application for the specific filing route already identified in this matter.

The application should seek:

1. Declaratory relief confirming that the relevant land rights form part of the estate of the late Moshe Shamash and pass to Sarina Shamash.
2. Mandatory relief directing the competent authorities to register, transfer, correct, annotate, or otherwise formalize those rights.
3. Any supplementary wording needed so the final order can actually be used before Tabu, RMI, or the relevant land-registration office.
4. Transfer to the competent forum rather than dismissal, if jurisdiction is challenged.

## Known facts to use

- Deceased: Moshe Shamash
- widow, heir, sole beneficiary: Sarina Shamash
- Probate basis exists: valid צו קיום צוואה
- Core property identifier: Gush 38049, Parcel 36, Subparcel 35
- Official property-status evidence exists tying Sarina Shamash to that land unit
- Authority correspondence states that registration cannot be completed without a court order

## Drafting requirements

- Write in formal, high-level Hebrew legal style
- Make the application serious, mature, and filing-ready
- Structure it as a real judicial application, not a casual letter
- Include heading, parties, jurisdiction, procedural background, facts, legal basis, requested relief, annex list, and signature block
- Use placeholders where details are missing
- Do not invent facts, statutes, case law, file numbers, parcel details, or evidence
- Keep the requested relief practical and operational for registration purposes

## Output wanted

- Main Hebrew judicial application
- Short version if needed
- Annex list
- Supporting affidavit template
- Short filing summary for portal upload

## Tone

Elite, precise, authoritative, justice-driven, and impossible to dismiss as vague or amateur.
